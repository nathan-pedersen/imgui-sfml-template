# ImGui-SFML Template
#### Dependencies
- SFML 2.6.2

#### Usage Linux (Maybe Mac too)
- Create a new repository from template.
- git clone --recurse-submodules \<your repo\>
- cd \<your repo\>
- mkdir build
- cd build
- cmake ..
- make
- run binary (./Main)

#### Usage Windows (Maybe Mac too)
- Create a new repository from template.
- git clone --recurse-submodules \<your repo\>
- cd \<your repo\>
- open directory in Visual Studio
